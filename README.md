# AI-Diagnostic-System-for-Disease-Classification-Using-Naive-Bayes-and-Decision-Tree-Classifiers
A hybrid machine learning diagnostic system that classifies infectious diseases (COVID19, Influenza, Dengue) from clinical symptoms and platelet count, combining probabilistic reasoning (Naive Bayes) with interpretable rule-based logic (Decision Tree). Data Used Synthetically generated diagnostic dataset of 300 patient records with 8 features: Fever, Cough, Fatigue, Headache, Rash, Loss of Smell, Joint Pain, Platelet Count Method Gaussian Naive Bayes + Decision Tree Classifier (max_depth=3), with 80/20 train-test split
Outcome Naive Bayes Accuracy: 85% | Decision Tree
Accuracy: 88.33%
